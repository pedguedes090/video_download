API Documentation

GET /api/video/info

Lấy thông tin video từ URL

curl -X GET "http://localhost:3000/api/video/info?url=VIDEO_URL"

POST /api/video/download

Tải xuống video với định dạng được chỉ định

curl -X POST "http://localhost:3000/api/video/download" \ -H "Content-Type: application/json" \ -d '{"url": "VIDEO_URL", "format": "mp4"}'

GET /health

Kiểm tra trạng thái server

curl -X GET "http://localhost:3000/health"
